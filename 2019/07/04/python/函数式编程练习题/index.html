sort和sorted的不同

  • 有无返回值

    • list.sort() ：对原列表进行排序，无返回值
    • sorted(list)：有返回值

  • 是否影响本身结构

    • sorted(list) ：不影响本身结构
    • sorted(list,reverse = True)：不影响本身结构
    • list.sort()：影响本身结构
    • list.sort(reverse = True)：影响本身结构
